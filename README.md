## âš™ï¸ Configuration

After installation, all agents are automatically available. No setup needed!

**Optional configuration:**

```json
{
  "hermetic-agents-pack": {
    "autoLoadAgents": true,
    "agentSelectionMode": "automatic",
    "verbosity": "normal"
  }
}
```

**Configuration Options:**

- `autoLoadAgents` - Load all agents on activation (default: true)
- `agentSelectionMode` - How agents are selected:
  - `automatic` - Claude chooses the best agent (default)
  - `manual` - You specify the agent
  - `hybrid` - Claude suggests, you confirm
- `verbosity` - Response detail level:
  - `minimal` - Concise responses
  - `normal` - Standard detail (default)
  - `detailed` - Comprehensive explanations
