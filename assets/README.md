# Hermetic Agents Pack - Assets

This directory contains visual assets for the Hermetic Agents Pack plugin.

## Directory Structure

### `/images`
General images, illustrations, and graphics for documentation and presentations.

### `/icons`
Icon files for the plugin and individual agents. These should follow the brand guidelines established in the main documentation.

### `/logos`
Official Hermetic Agents Pack logos in various formats and sizes.
- Primary logo (full color)
- Secondary logo variations
- Monochrome versions
- Different file formats (SVG, PNG, etc.)

### `/screenshots`
Screenshot examples demonstrating agent capabilities and plugin usage.
- Agent interaction examples
- Configuration screenshots
- Usage demonstrations

## Brand Guidelines

All assets should follow the brand guidelines documented in:
- `STYLE-GUIDE.md` - Voice and tone
- `BRAND-GUIDELINES.md` - Visual identity

## File Naming Conventions

Use clear, descriptive names:
- `agent-[name]-icon.png` - For agent-specific icons
- `logo-primary-[version].svg` - For logo variations
- `screenshot-[feature]-[date].png` - For screenshots
- `banner-[usage]-[size].png` - For banner images

## Recommended Formats

- **Logos**: SVG (vector), PNG (raster backup)
- **Icons**: SVG, PNG with transparency
- **Screenshots**: PNG for clarity
- **General Images**: PNG or JPG as appropriate
