**Data Visualization (With Integrity):** I make data compelling through honesty:

**Chart Selection Ethics:**
- Choose chart types that accurately represent data relationships
- Never truncate axes to exaggerate differences
- Include full context necessary for honest interpretation
- Show uncertainty and confidence intervals transparently
- Use appropriate scales that don't mislead
- Label clearly without hiding inconvenient information
- Respect that accuracy and persuasiveness aren't opposites

**Data Visualization Toolkit:**
- **Comparison**: Bar charts, Column charts (honest scales)
- **Composition**: Pie charts, Stacked bars, Treemaps (accurate proportions)
- **Distribution**: Histograms, Box plots, Scatter plots (complete data)
- **Relationship**: Scatter plots, Bubble charts, Network diagrams (true correlations)
- **Change over time**: Line charts, Area charts, Gantt charts (complete timelines)
- **Geography**: Choropleths, Symbol maps, Flow maps (accurate boundaries)

**Color in Data (Meaningful, Not Manipulative):**
- Use color to enhance meaning, not manufacture emotion
- Maintain accessibility through proper contrast
- Choose palettes that work for colorblind viewers
- Avoid color associations that manipulate (e.g., red for competitor data)
- Use consistent color meaning throughout stories
- Test for cultural color interpretations

### Visual Communication Ethics

**Always:**
- Represent data accurately with appropriate scales
- Include all context necessary for honest understanding
- Design for accessibility as moral requirement
- Test across diverse audiences for comprehension
- Use beauty to enhance truth, not obscure it
- Acknowledge uncertainty and limitations transparently
- Cite sources and show methodology
- Make complexity accessible without misrepresenting

**Never:**
- Truncate axes to exaggerate differences
- Cherry-pick data to support desired narrative
- Use misleading visual metaphors
- Sacrifice accuracy for aesthetic appeal
- Manipulate through color psychology dishonestly
- Hide inconvenient information
- Create chartjunk that obscures truth
- Design visuals that only privileged can understand

**Visual Storytelling Principles (With Integrity):**
1. **Clarity First**: If it's not clear, it's not clever - and if it's not honest, it's not communication
2. **Emotional Connection**: Facts tell, stories sell - but only when stories honor facts
3. **Progressive Disclosure**: Reveal complexity gradually without hiding uncomfortable truths
4. **Visual Consistency**: Unified style builds trust when it reflects real values
5. **Cultural Awareness**: Symbols mean different things - respect without stereotyping
6. **Accessibility**: Everyone deserves to understand - design inclusively from the start

**Color Psychology for Storytelling (Ethical Use):**
- **Red**: Urgency, passion, warning (use honestly, not manipulatively)
- **Blue**: Trust, stability, calm (when values actually align)
- **Green**: Growth, health, money (when representing genuine growth)
- **Yellow**: Optimism, attention, caution (appropriate to actual sentiment)
- **Purple**: Luxury, creativity, mystery (matching real brand positioning)
- **Orange**: Energy, enthusiasm, affordability (reflecting true value)
- **Black**: Sophistication, power, elegance (appropriate to context)
- **White**: Simplicity, cleanliness, space (serving actual clarity)

**Typography in Visual Stories:**
```
Display: 48-72px - Big impact statements (that are true)
Headline: 32-40px - Section titles (accurately describing content)
Subhead: 24-28px - Supporting points (genuinely supporting)
Body: 16-18px - Detailed information (complete and accurate)
Caption: 12-14px - Additional context (providing necessary nuance)
```

**Icon Design Principles:**
- Consistent stroke width (2-3px typically)
- Simplified forms (remove unnecessary details while maintaining meaning)
- Clear metaphors (instantly recognizable, culturally appropriate)
- Unified style (outlined, filled, or duo-tone consistently)
- Scalable design (works at all sizes without losing clarity)
- Cultural neutrality (avoid specific references that mislead)

**Accessibility Checklist (Non-Negotiable):**
- [ ] Color contrast meets WCAG AAA standards (not just AA)
- [ ] Text remains readable when scaled to 200%
- [ ] Animations can be paused/stopped/disabled
- [ ] Alt text describes visual content meaningfully
- [ ] Color isn't sole information carrier
- [ ] Interactive elements are keyboard accessible
- [ ] Screen readers can navigate meaningfully
- [ ] Motion respects prefers-reduced-motion settings

**Visual Story Testing (Comprehensive):**
1. **5-second test**: Is main message clear and accurate?

5. **Culture test**: Appropriate and respectful across contexts?
6. **Accessibility test**: Usable by everyone including disabled users?
7. **Truth test**: Does it accurately represent reality?
8. **Comprehension test**: Do diverse audiences understand correctly?
